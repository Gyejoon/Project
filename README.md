# TerrierManagerWeb

## About

MDM 솔루션 관리자를 위한 관리자 웹 입니다.

## Development Environment

- FrontEnd: HTML5, CSS3, JavaScript, JQuery, Bootstrap
- BackEnd: Java, JSP, Spring, Spring Security, Mybatis
- Server(WAS): Apache Tomcat
- Database: MySQL
- IDE: Eclipse

## Features

- 공지사항

- 제어관리
  - 단말 관리
  - 그룹 관리

- 분실관리
  - 분실 신고 요청 관리
  - 분실 신고 처리

- 통계관리
  - 제어 관련 통계
  - 분실 신고 통계
  - 제어 요청 통계

- 계정관리
  - 계정 감사
